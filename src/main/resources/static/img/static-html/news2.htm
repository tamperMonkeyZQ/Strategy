省委、省政府召开辽宁沿海经济带落实 “三大区域发展战略”工作座谈会

发布时间：2017/12/29

12月8日，省委、省政府在辽宁人民会堂召开辽宁沿海经济带落实“三大区域发展战略”工作座谈会。会议深入学习贯彻党的十九大精神，沿海六市、省直有关部门围绕落实全省推进“三大区域发展战略”工作电视电话会议精神，汇报了工作进展情况和明年工作安排。省委书记陈求发出席会议并讲话，省委副书记、代省长唐一军主持会议并讲话。省发展改革委吴忠琼主任做了工作汇报。

省委书记陈求发指出，要遵循2013年习近平总书记在辽宁考察时确定的，沿海经济带的发展目标定位、功能和发展路径，深入学习贯彻党的十九大精神，确保如期完成辽宁沿海经济带三年攻坚计划，努力把辽宁沿海经济带打造成辽宁经济重要增长极，并提出要集中精力抓好的六个方面的工作。一是抓紧制定2018年工作计划；二是要在产业协同上下功夫；三是要坚定不移扩大对外开放；四是要大力深化改革；五是加快推进港口资源整合；六是建立健全沟通协调机制。 　　

代省长唐一军指出，以贯彻党的十九大精神为动力，以更加开放的姿态、更加宽广的视野、更加有力的举措、更加扎实的工作，加快推进沿海经济带建设，努力打造践行新发展理念的产业结构优化先导区、经济社会发展先行区，成为带动辽宁新一轮振兴发展的发动机。一要强化开放为先，引领振兴发展。二要强化创新为重，培育发展动能。三要强化统筹为要，推动协调发展。四要强化绿色为求，建设美丽辽海。五要强化民生为本，创造美好生活。 　　

忠琼主任代表省发展改革委作了汇报，重点就省发展改革委深入沿海经济带各市、县（区）和重点产业园区，宣讲三年攻坚计划、指导各市编制三年攻坚计划及积极组织沿海各市申报项目等工作情况作了介绍。
