全省教育服务五大区域发展战略和一带五基地建设工作推进会在沈阳召开

发布时间：2018年5月16日

为贯彻落实省委、省政府关于五大区域发展战略和一带五基地建设工作的统一部署，充分发挥全省优势教育资源在地区经济社会发展中的服务保障和支撑引领作用，5月15日，全省教育服务五大区域发展战略和一带五基地建设工作推进会在沈阳召开。省委高校工委副书记，省教育厅党组书记、厅长唐国华出席会议并讲话。

唐国华指出，“五大区域发展战略”和“一带五基地”，是中央和省委经过慎重考虑，就辽宁全面建成小康社会和老工业基地新一轮振兴做出的重大决策部署，是我们建设社会主义现代化强国的重要组成部分，是辽宁全面振兴发展的必由之路。全省教育系统一定要提高认识，加强领导，统一思想，凝聚共识，切实将思想和行动统一到中央和省委的决策部署上来。

唐国华强调，全省教育系统切实要增强做好各项服务工作的责任感和使命感，主动将服务“五大区域发展战略”和“一带五基地”建设作为深化教育领域综合改革的重要突破口，奋发有为，扎实工作，努力开创教育服务辽宁振兴发展的新局面，重点要抓好七方面的工作。一是提高认识，进一步增强做好服务“五大区域发展战略”和“一带五基地”建设工作的责任感使命感；二是加强领导，进一步完善工作机制；三是精心谋划，认真制定服务振兴发展的实施方案；四是深化产教融合，进一步增强教育服务振兴发展的能力；五是加强应用研究，多渠道加快科技成果转化；六是搭建信息共享服务平台，全面做好服务对接工作；七是加强绩效考核，加大监督管理考核力度。

沈阳化工大学、沈阳农业大学、辽宁科技大学等3所学校的领导同志就本校服务辽宁振兴发展工作作了典型发言，从不同侧面介绍了学校在服务五大区域发展战略和一带五基地建设方面所做的工作和取得的成绩。

会议还对教育服务“五大区域发展战略”和“一带五基地”建设工作信息共享平台的使用举办了专题培训。各市教育局、全省高校及相关部门、处室负责同志参加会议。

（工业高等教育处 供稿）
